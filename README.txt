 Auteur : Thomas Chaussende
 Date dernière mise a jour : 12/05/2024
 Nom du programme : Balles Rebondissantes

ce projet est divisé en 3 parties différentes a partir d'un menu principal:
- Balles rebondissantes:
	Projet ayant pour but de faire apparaitre des balles qui rebondit sur les murs 
	et qui possède une image de pierre, feuille ou ciseaux, si elles rentrent en
	contact elles font le pierre feuille ciseaux et le perdant prend le choix du gagnant
- Animation :
	Mon animation est un jukebox qui propose d'écoute entre 8 musique de mes jeux préférer
- Object 3D :
	Projet qui montre un objet en 3D tourner sur l'axe X
